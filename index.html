Descarga gratuita de B++

Obtenga el lenguaje que potencia la transpilación nativa en dispositivos móviles y sistemas industriales.

¿Por qué usar B++?

B++ permite optimizar flujos de trabajo mediante transpilación cruzada de alto nivel. Al descargar B++, accede al núcleo PattitexTEC, diseñado para ofrecer velocidad nativa con sintaxis moderna.

¿Cómo usar B++?

B++ usa una sintaxis simple enfocada en terminales móviles como Pydroid 3 y Termux. Puede crear variables, mostrar texto, hacer preguntas, usar colores y ejecutar condiciones.

Ejemplo básico

{
  val nombre = "Doryan"
  text("Hola") -> VERDE
  text(nombre)
}

Comandos principales

  • val → Crear variables
  • text() → Mostrar texto en consola
  • preguntar() → Pedir datos al usuario
  • si → Ejecutar condiciones
  • esperar() → Pausar ejecución
  • limpiar → Limpiar terminal

Entrada de usuario

{
  preguntar("Como te llamas?") -> nombre
  text(nombre) -> CYAN
}

Condiciones

{
  val admin = "si"
  si admin == "si" -> text("Acceso permitido") -> VERDE
}

Colores disponibles

ROJO, VERDE, AZUL, AMARILLO, CYAN, MAGENTA y GRIS.

Compilación

  • Python (.py)
  • C++ (.cpp)
  • Binario (.bin)

Estructura recomendada
